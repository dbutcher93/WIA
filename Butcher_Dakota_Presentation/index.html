Why I chose this plugin

I looked at over 100 plugins and so many of them seemed fun to do. I originally was going to do my presentation on Unslider, which is a content slider plugin. After a few attempts I decided to try Remodal and it was very intuitive and fun to use.

How I started the project

I started this project with the revealJs files. I started reserching the plugin, trying to figure out how to set up Remodal. As I said before this plugin is very intuitive, so it was pretty easy to learn.

Code View


<div class="remodal-bg">
    <a href="#modal"><h2>Click Here</h2></a>
    <div class="remodal" data-remodal-id="modal">
        <h1>Example</h1>
        <p>This is my example modal.</p>

        <a class="remodal-confirm" href="#">OK</a>
    </div>
</div>

Code Setup

As you can see in the code view it did not take much code to make this possible. All it takes is a div with the class of "remodal-bg", then you need a anchor tag to call the modal. After all of that, the only thing left is to add the modal. To do that you just add a div with a class of "remodal", add your content and you cand add a confirm or cancel button to the modal.

Wrap Up

I had a lot of fun with this plugin. It would be very usful if you do not have the knowledge to make a modal yourself, or even if your using it on a personal project. All of the plugins that I looked into looked about as intuitive as this one and I will look more into plugins in the future.
